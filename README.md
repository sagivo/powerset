powerset
========

ruby implementation for finding the powerset of set. 
arr = [1,2,3,4]
powerset arr 
#[[], [1], [2], [1, 2], [3], [1, 3], [2, 3], [1, 2, 3], [4], [1, 4], [2, 4], [1, 2, 4], [3, 4], [1, 3, 4], [2, 3, 4], [1, 2, 3, 4]]

usage
========

link this powerset to your code and simple call `powerset som_array`
